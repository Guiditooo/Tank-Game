# Tank-Game
 
- Description: A game where you have to shoot different targets driving a little tank.

- Controls: W - Gas
			S - Break/Reverse
			A - Turn Left
			D - Turn Right
			Right Clic (pressed) & mouse movement - Camera Control
			Left Clic - Move The Cannon & Shoot.
